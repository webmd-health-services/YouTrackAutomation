# System Requirements

* Windows PowerShell 5.1 and .NET 4.6.1+
* PowerShell Core 6+

# Installing

To install globally:

```powershell
Install-Module -Name 'YouTrackAutomation'
Import-Module -Name 'YouTrackAutomation'
```

To install privately:

```powershell
Save-Module -Name 'YouTrackAutomation' -Path '.'
Import-Module -Name '.\YouTrackAutomation'
```
